📚 Exponential Function Properties

✔ General form: y = a^x (a > 0, a ≠ 1)

✔ Always passes through (0, 1)

✔ (1, a) shows the base directly

✔ (1/a, 1) shows inverse symmetry idea

✔ If a > 1 → exponential growth 📈

✔ If 0 < a < 1 → exponential decay 📉

✔ Domain: all real numbers

✔ Range: y > 0 always
