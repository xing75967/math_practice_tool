📊 Power Function vs Exponential Function

1. Definitions

Power Function: y = x², y = x³

Exponential Function: y = 2ˣ

2. Key Idea Difference

Power function: x is changing, exponent is fixed → growth depends on input size

Exponential function: exponent is changing → growth depends on repeated multiplication

3. Graph Comparison

Red: y = x²
Purple: y = x³
Blue: y = 2ˣ

4. Most Important Visual Difference

  • Power function grows symmetrically or polynomially
  • Exponential function grows slow → then explodes rapidly
  • Exponential eventually dominates ANY power function

5. Why Does Exponential Dominate?

Meaning: Exponential eventually dominates ANY power function means:

👉 No matter how big the power is (x², x³, x¹⁰⁰...), the exponential function (like 2ˣ) will become larger when x is big enough.

Example Comparison:

x = 5 → x² = 25, x³ = 125, 2ˣ = 32

x = 10 → x² = 100, x³ = 1000, 2ˣ = 1024

x = 20 → x² = 400, x³ = 8000, 2ˣ = 1,048,576 🚀

Key Insight:

Power functions grow step by step.

Exponential functions grow by multiplying again and again.
