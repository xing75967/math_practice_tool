📈 Quadratic Function Summary

✔ General form: y = ax² + bx + c

✔ a controls opening direction & width

✔ b controls horizontal shift (vertex movement)

✔ c is the y-intercept (point at x = 0)

✔ If a > 0 → opens upward 📈

✔ If a < 0 → opens downward 📉

✔ x = -b / (2a) is the symmetry axis

✔ Vertex calculation: （ -b / (2a), (4ac-b²) / (4a) ）

✔ Vertex is the turning point of the parabola

✔ Vertex moves continuously as parameters change
