GMAT Parity Rules (Odd & Even)

Rule 1:
Even = Even + Even OR Odd + Odd
Even = Even × Even OR Odd × Even
4 + 6 = 10 (Even)
3 + 5 = 8 (Even)
2 × 6 = 12 (Even)
3 × 4 = 12 (Even)
Rule 2: Odd = Odd + Even
3 + 4 = 7 (Odd)
Rule 3: Sum of an odd number of odd integers is odd
3 + 5 + 7 = 15 (Odd)
Rule 4: Sum of an even number of odd integers is even
3 + 5 = 8 (Even)
Rule 5: Any number of even integers added/subtracted is even
2 + 4 + 6 = 12 (Even)
Rule 6: If a product of n integers is odd, then all must be odd
3 × 5 × 7 = 105 (Odd)
If any number is even → result becomes even
Rule 7: If n consecutive integers sum to 0, then n is odd
(-2) + (-1) + 0 + 1 + 2 = 0
Count = 5 (odd)
Rule 8: If n consecutive odd integers sum to 0, then n is even
(-3) + (-1) + 1 + 3 = 0
Count = 4 (even)
Rule 9: If n consecutive even integers sum to 0, then n is odd
(-4) + (-2) + 0 + 2 + 4 = 0
Count = 5 (odd)
Rule 10: If the sum of two prime numbers is odd, one of them must be 2
2 + 3 = 5 (Odd)
3 + 5 = 8 (Even) ❌
⚠️ Trap: Students forget that ALL primes except 2 are odd!
